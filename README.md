# Hackathon Project

A full-stack hackathon application with Python backend (FastAPI/Django), React/Vue frontend, and Supabase database.

## Quick Start

**For the first time setup**, see [SETUP.md](SETUP.md) for detailed instructions.

**For continuing development**:

```bash
# Activate environment
source .venv/bin/activate

# Start backend
uv run python backend/app/main.py

# In another terminal, start frontend (from frontend/)
npm run dev
```

## Tech Stack

- **Backend**: Python 3.10+ with FastAPI/Django
- **Frontend**: Node.js 18+ with React/Vue + Vite
- **Database**: PostgreSQL (Supabase)
- **Code Quality**: ruff, mypy, biome, pre-commit
